# Practic1 Project

## Описание
Простейший проект на Go с функциями Add, Mult, Pow, Div, Sub.

## Функции
- `Add(a, b int) int` — сложение двух чисел
- `Mult(a, b int) int` — умножение двух чисел
- `Pow(base, exp int) int` — возведение числа в степень
- `Sub(a, b int) int` — вычитание двух чисел
- `Div(base, exp int) int` — деление двух чисел
